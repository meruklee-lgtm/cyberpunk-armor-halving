# Cyberpunk RED - Armor Halving

Ein Foundry VTT Modul für Cyberpunk RED, das automatisch alle Panzerungswerte (SP) halbiert.

## Installation über The Forge

1. Kopiere diese Manifest-URL in Foundry VTT unter "Install from Manifest":
```
https://raw.githubusercontent.com/YOUR-USERNAME/cyberpunk-armor-halving/main/module.json
```
(Ersetze YOUR-USERNAME mit deinem GitHub Benutzernamen)

2. Aktiviere das Modul in deiner Welt unter "Game Settings" → "Manage Modules"

## Funktionen

- ✅ Halbiert automatisch alle SP-Werte von Body und Head Location
- ✅ Optional: Halbiert auch Schild Hit Points
- ✅ Ein-/Ausschaltbar über Modul-Einstellungen
- ✅ Keine permanenten Änderungen an Items
- ✅ Funktioniert für alle Charaktere automatisch

## Einstellungen

Im Modul-Menü kannst du konfigurieren:
- **Panzerung halbieren**: Aktiviert/Deaktiviert die Halbierung aller Panzerungen
- **Schilde auch halbieren**: Halbiert auch ballistische Schilde (optional)

## Wie es funktioniert

Das Modul modifiziert die Panzerungswerte dynamisch beim Laden, ohne die Original-Items zu verändern. Wenn du das Modul deaktivierst, sind alle Werte wieder normal.
